Company : Axis Bank
Company Name : Axis Bank
Nature Of Business : Banking
Designation : Manager IT
Description :

Tentative Job Location

Mumbai / Bangalore

 

Job Description

 

Responsibilities

  • Execute HR policies and procedures relevant to IT (Recruitment, Training, Evaluation, Promotion, Release etc.) for the team
  • Translate business requirements into high-level design specification
  • Determine effort & impact out of business requirement (mandatory for Project Initiation Document)
  • Approve decisions with regards to application platform
  • Build and review implementation plans (of solutions)
  • Define service level agreements (SLAs) for application services
  • Define operating level agreements (OLAs) for meeting SLAs [Applications]
  • Define operating level agreements (OLAs) for meeting SLAs  [Infrastructure]
  • Monitor application service level performance
  • Review SLAs and underpinning contracts for application services
  • Conduct business impact analysis and risk assessment for applications
  • Review status of application related problems
  • New technical requirements are properly integrated with existing processes
  • Participate during requirement gathering
  • Write technical requirements from a critical phase
  • Interact with vendors to understand software limitations
  • Assist programmers during system development, ex: provide use cases, flowcharts or even database design
  • Responsible for system testing (SIT, Performance, security testing)
  • Responsible for designing components and providing that information to the developer.
  • Monitoring system compliance in accordance with IT policy standards.
  • Responsible for operational and Maintenance activities of the system
  • Evaluate alternative solutions to business problems and select preferred solution. Justify selection.

 

Deliver & Support

  • Resolve, recover and close application related incidents
  • Identify and classify application/data-warehouse related problems
  • Perform root cause analysis for application related problems
  • Resolve application related problems
  • Maintain application related problem records, known problems, known errors and workarounds

 

Specific Technical Skills:

  • Java skills (mandatory)
  • java security (cryptography, authentication and authorization)
  • java collection, multithread, IO, webservice, jsp, servlet, javascript
  • html5, ejb, struts, hibernate would be an added advantage
  • .net skills (preferred, but not mandatory)
  • C, C#, .NET, ASP.NET
  • Web Services, SOAP, XML, XSL, XSLT
  • DTS, SSRS, SSIS would be an added advantage
  • Database Skill common for both java and .net
  • Oracle, Sql-Server, writing SQL and PL/SQL
